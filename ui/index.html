TinyLLM Architecture

A completely custom, causal language model built from scratch.

The Mathematics

The complete inference step of the network, from token IDs to output logits, can be summarized mathematically by the following consolidated equations:

$$ \begin{aligned} H_0 &= \text{Embed}(X) \\ H_l &= F_l(H_{l-1}) \quad \text{for } l \in \{1, 2, \dots, N\} \\ \text{Logits}(X) &= \text{RMSNorm}_{\text{final}}(H_N) W_{\text{head}}^T \\ F_l(H) &= \tilde{H} + \left[ \text{SiLU}\left( \text{RMSNorm}_{\text{ffn}}(\tilde{H}) W_1^T \right) \odot \left( \text{RMSNorm}_{\text{ffn}}(\tilde{H}) W_3^T \right) \right] W_2^T \\ \text{where} \quad \tilde{H} &= H + A_l\big(\text{RMSNorm}_{\text{attn}}(H)\big) \\ A_l(x) &= \text{softmax}\left(\frac{\text{RoPE}(x W_q^T)\text{RoPE}(x W_k^T)^T}{\sqrt{d_{\text{head}}}} + M\right)(x W_v^T) W_o^T \end{aligned} $$

Implementation Details

  • Rotary Positional Embeddings (RoPE): Injects relative positional information directly into the attention mechanism by rotating the query and key representations.
  • RMSNorm (Root Mean Square Normalization): Used instead of standard LayerNorm. It removes the mean-centering step for computational efficiency. Applied in a "Pre-Norm" fashion before Attention and FFN blocks.
  • SwiGLU Activation: The Feed-Forward Network uses the Swish-Gated Linear Unit, utilizing the element-wise Hadamard product to gate information flow.
  • Byte-Pair Encoding (BPE): A custom Python tokenizer that parses raw text and merges adjacent character pairs based on corpus frequency.
